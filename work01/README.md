## 📌 作业目标
基础图像操作：
1. 使用OpenCV读取一张测试图片 
2. 输出图片基本信息
3. 显示原图：用Matplotlib显示图片
4. 转换为灰度图并显示
5. 保存灰度图
6. 用NumPY做操作

## 📂 项目结构
work01/
├── code/                  # 代码目录
│   └── task.py            # 主程序：实现图片读取、灰度转换、像素修改
└── images/                # 图片资源目录
    ├── test.jpg           # 输入原始图片
    ├── test_gray.jpg      # 输出：灰度化后的图片
    └── pixel_change.jpg   # 输出：像素修改后的图片
└── README.md

## 🚀 运行说明
1. 确保已安装 OpenCV 库：`pip install opencv-python`
2. 进入 `code/` 目录，执行：`python task.py`
3. 运行后会在 `images/` 目录生成处理后的图片文件

## 📝 备注
所有图片处理结果均保存在 `images/` 目录下，代码中已配置好相对路径，直接运行即可生成结果。
