本实验实现图像1/2下采样、三种插值恢复，并通过MSE/PSNR、FFT、DCT进行质量与频域分析，对比不同方法效果。
一、实验目的
掌握图像1/2下采样的两种实现：直接下采样、高斯平滑后下采样，理解混叠与滤波作用。
熟悉最近邻、双线性、双三次三种插值方法，将缩小图恢复为原图尺寸并对比效果。
使用MSE、PSNR量化评价恢复质量，结合FFT、DCT分析高低频变化。
分析不同恢复方法的DCT能量分布，总结下采样与插值规律。

二、实验原理
2.1、图像下采样
将图像按比例缩小，本次为1/2下采样。
直接下采样：直接抽取像素，易出现混叠失真、细节模糊。
高斯平滑后下采样：先做低通滤波，再缩小，可减少混叠，但会损失部分细节。
2.2、图像插值恢复
将缩小图放大回原图尺寸，填充缺失像素：
最近邻：复制最近像素，速度快，但有锯齿、块效应。
双线性：用周围4个像素加权计算，图像更平滑。
双三次：用周围16个像素拟合，边缘更清晰，效果最好。
2.3、质量评价指标
MSE：均方误差，数值越小，图像越接近原图。
PSNR：峰值信噪比，数值越大，图像质量越好。
2.4、FFT频域分析
将图像转为频域：
低频：整体轮廓、亮度。
高频：边缘、细节、纹理。
可直观看到细节丢失与混叠情况。
2.5、DCT变换分析
能量主要集中在左上角低频区域。统计左上角1/4低频能量占比：占比越高，图像越平滑，细节越少。

三、实验步骤
读取灰度图像，获取尺寸。
两种方式下采样：直接缩小、高斯模糊后缩小。
三种插值将缩小图恢复原图尺寸。
计算MSE、PSNR并打印。
对原图、缩小图、恢复图做FFT，保存频谱图。
对原图及恢复图做DCT，计算低频占比并保存DCT图。
绘制并保存空间域对比图。

五、结果分析
5.1、空间域指标
双线性：MSE最小、PSNR最高，指标最优。
双三次：指标略低，但视觉边缘更清晰、更接近原图。
最近邻：指标最差，有明显锯齿和块效应。
5.2、FFT高频成分分析
原图：高频丰富，细节完整。
缩小后：高频大量丢失。
恢复后：高频远少于原图，插值无法找回丢失细节。
5.3、DCT能量分布分析
双线性：低频占比最高，最平滑，细节丢失最多。
双三次：低频占比最接近原图，细节保留最好。
最近邻：块效应带来额外高频，低频占比略低。

六、实验结论
直接下采样效果稍好，但会混叠；高斯下采样防混叠，但损失细节。
恢复效果：双三次≈双线性＞最近邻。
插值只能填充像素，无法重建下采样丢失的高频细节。
FFT和DCT能清晰反映图像细节丢失与平滑程度。
