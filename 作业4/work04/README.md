## 实验内容
用Sobel算子计算图像梯度，估计局部下采样倍数M
对不同区域用对应σ（σ=0.45M）做高斯滤波
分块自适应下采样，与全图统一下采样做误差对比

## 实验原理
1. 图像下采样与混叠
下采样：通过隔像素取点降低图像分辨率
混叠：直接下采样时，图像中的高频分量（如棋盘格边缘、chirp 图高频区域）会因不满足奈奎斯特采样定理，混叠到低频区域，产生失真
抗混叠：下采样前先做高斯低通滤波，滤除高于奈奎斯特频率的分量，再下采样可消除混叠

2. 高斯滤波σ参数公式
对于下采样倍数M，理论最优高斯标准差为：σ=0.45×M
原理：σ太小→滤波不足，混叠残留；σ太大→过度模糊，细节丢失。σ=0.45M是兼顾抗混叠与细节保留的最优值。

3. 自适应下采样
核心思想：根据图像局部细节（梯度）动态调整下采样倍数：梯度大（细节/边缘多→小下采样倍数（保留细节），梯度小（平滑区域→大下采样倍数（压缩数据）

## 实验结论
直接下采样会产生严重混叠，高斯滤波后下采样可有效消除混叠
高斯滤波σ=0.45M（M为下采样倍数）是兼顾抗混叠与细节保留的最优值
基于梯度的自适应下采样，在细节区域保留效果远优于统一下采样，整体误差更低，实现了更高效的图像压缩
