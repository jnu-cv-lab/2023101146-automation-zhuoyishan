## 五、必须回答的问题
1. 
特征点是图像中具有唯一性、稳定性、可重复性的局部关键位置，如角点、边缘交点、纹理突变点，能在视角、光照、尺度变化时被稳定检测。
1)主要分布在角点、文字边缘、图案纹理丰富、亮度突变的区域。在box.png中，特征点集中在包装盒文字、图案、棱角处。
2)这些区域灰度变化剧烈、方向信息明确、唯一性强，FAST角点检测器容易判定为稳定特征点。
3)平坦区域灰度几乎无变化，没有角点、边缘和纹理差异，无法提取稳定特征。
2. 
特征描述子是表征关键点局部外观的数值向量，用于衡量两个特征的相似性。
1)关键点：记录位置、方向、大小等几何信息。描述子：记录关键点周围的视觉特征，用于匹配。
2)只知道位置无法判断两幅图的点是否对应，描述子提供可量化的相似度，才能实现正确匹配。
3)ORB描述子为32维（256位二进制）。
4)因为它由二进制比特串0/1组成，由BRIEF改进而来，生成快、匹配快、占用内存小。
3. 
1)由256位二进制串构成，每一位表示关键点邻域内两次像素亮度比较结果。
2)衡量两个二进制串对应位不同的数量，值越小表示越相似。
3)欧氏距离用于浮点数向量。ORB是二进制，只能用汉明距离，计算极快。
4. 
1)描述子在关键点局部小窗口计算，整体平移不影响局部窗口特征。
2)为每个关键点计算主方向，所有描述子都按主方向对齐，实现旋转不变。
3)构建图像金字塔（多层不同分辨率图像），在不同层级检测特征，适应大小变化。
4)不能。ORB仅保证局部特征稳定，但大角度透视会导致局部外观彻底改变，单靠特征无法完全抵抗，必须依靠Homography与RANSAC。

3)不一定。相似区域会产生距离小但位置错误的匹配，必须靠RANSAC进一步过滤。
